# Thursday Football Club Management System

A modern football club management system built with Next.js and Supabase.

## Features
- Player registration and management
- Match scheduling and results
- Club dashboard
- Authentication system

## Getting Started
```bash
npm install
npm run dev
```

Visit [thursdayfootball.com](https://thursdayfootball.com) to see it live!
